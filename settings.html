How it works:
UltraVPN Pro fetches Mullvad's public relay list from api.mullvad.net/www/relays/all/ and caches it locally for 6 hours. When you connect to a country, it picks the best available bridge server (Mullvad's native SOCKS5 relay type) for that country. Your Mullvad account number is used for SOCKS5 authentication — it's stored locally and never transmitted anywhere except to the Mullvad proxy server at connection time.

Bridge servers are Mullvad's dedicated SOCKS5 proxy endpoints. When available for a country they appear as ⬡ N next to the country name in the server list.
⚠️ A valid Mullvad subscription is required. This extension routes only browser traffic through the proxy — for system-wide VPN use the Mullvad desktop app.
